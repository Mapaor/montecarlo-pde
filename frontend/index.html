Monte Carlo PDE Solver

About

This application solves Partial Differential Equations (Poisson and Laplace equations) using Monte Carlo methods. The Monte Carlo approach uses random walks to approximate the solution at a given point, making it particularly useful for complex boundary value problems.

  • Laplace Equation: ∇²φ = 0 (no charge density)
  • Poisson Equation: ∇²φ = -ρ/ε₀ (with charge density)
